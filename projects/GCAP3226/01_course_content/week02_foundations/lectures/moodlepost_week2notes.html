💻 Technical Setup: VS Code & Jupyter Notebooks

Understanding the Tools

  • Python Environment (virtual): Like a restaurant (real environment) 🍽️
  • Jupyter Notebook: Like the dining area where you write the recipe (code) of your dish (task) 📝
  • Kernel: Like the restaurant manager coordinating everything 👨‍💼
  • Interpreter: Like the chef prepares for the dish (executes code) 👨‍🍳
  • Python Libraries: Like ingredients for cooking 🥕

🚀 Quick Setup Steps

  1. Install Libraries: Open Terminal/Command Prompt
  2. Run command:
pip install jupyter pandas matplotlib seaborn
  1. Install Jupyter Extension in VS Code
  2. Choose Python Environment when prompted for kernel

🤖 GitHub Copilot: Your AI Programming Partner

How to Work with AI

  • Write Clear Prompts: Use comments like # Write code to create a bar chart for support levels
  • Review AI Suggestions: Understand what the code does before running
  • Iterate and Improve: Ask AI to modify code for better visualization
  • Learn by Doing: Don't worry about understanding every detail initially

Remember: You can treat AI-generated code as a "black box" initially. Focus on asking the right questions and interpreting the results for policy insights!
