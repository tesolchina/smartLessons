🚌🤝 Inter-Company Bus Route Coordination Project

GCAP 3226: Empowering Citizens through Data

Topic 3 • Group Work (4-6 students) • Multi-Operator Coordination Analysis

🎯 Project Overview

Analyze how competing bus operators (KMB Route 272A and CityBus Route 582) could coordinate schedules to optimize passenger experience at their 8 overlapping stops.

This project offers a unique opportunity to evaluate data governance in competitive transportation markets. You'll examine how different operators could coordinate timetables to reduce passenger waiting times while maintaining business viability, using real-time API data and advanced scheduling algorithms.

✅ Dual-API System Built: Working Jupyter notebook with real-time KMB and CityBus API integration

✅ Real-Time Data Collection: Concurrent thread-based ETA monitoring with automatic data merging

❓ Implementation Framework: How might real-time coordination be managed using existing technological infrastructure?
