🚏 Bus Stop Merger Optimization Project

GCAP 3226: Empowering Citizens through Data

Topic 2 • Group Work (4-6 students) • Advanced Urban Planning Analysis

🎯 Project Overview

Evaluate whether merging adjacent bus stops improves public transport efficiency and accessibility, using St. Martin Road and Chong San Road stations as a case study.

This project offers an exciting opportunity to evaluate data governance practices in urban transport planning. You'll analyze how proximity affects service efficiency when two bus stops are only 52 meters apart, examining passenger accessibility, operational costs, and traffic flow impacts through advanced simulation modeling.

🚀 Head Start: Advanced Research Foundation

✅ Comprehensive Dissertation Completed: Master's-level research with Python-based discrete-event simulation analysis
✅ Real Data Collected: KMB Open Data API integration with actual bus schedules and stop locations
✅ Working Code Available: Jupyter notebook with operational simulation models and queuing theory implementation
✅ Four Scenarios Analyzed: Current setup, designated routes, merged superstation, and single-station configurations

📈 Key Research Questions

❓ Optimal Distance Analysis: What factors might determine the ideal spacing between consecutive bus stops?
❓ Accessibility Trade-offs: How might stop mergers differentially impact various user demographics and mobility needs?
❓ Policy Framework Development: What decision-making criteria could potentially guide future stop merger evaluations?
❓ Implementation Strategy: How might simulation insights be translated into practical urban planning policies?
