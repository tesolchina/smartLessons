🎯 Project Overview

Analyze Hong Kong Observatory's typhoon signal decision-making using real-time wind data from 30 weather stations, examining the accuracy and transparency of public warning systems.

This project offers an extraordinary opportunity to evaluate data governance in emergency public communications. You'll examine discrepancies between published signal criteria and actual wind measurements, building upon comprehensive API integration and real-time monitoring systems already deployed.
