Video Transcript with Key Points Highlighted

In this video you will learn how to properly cite Journal articles in the APA 7th edition Style. By the end of this video you will be able to reference a journal article using in-text citations and reference lists as well as identify best practices for doing so.

APA style referencing involves two key elements: in-text citations within the body of your paper and a comprehensive reference list at the end.

In-Text Citations: Whenever you reference a journal article in the text of your paper you'll need to include an in-text citation. The basic elements of an in-text citation are the author's surname and the year of publication.

For example, if the author's name is John Smith and the publication year is 2024, the in-text citation would be Smith 2024.

Two Types of Citations:

• Parenthetical citation: you enclose the author's surname and publication year in parentheses

• Narrative citation: you incorporate the author's name into the sentence and then include the publication year in parenthesis

Reference List Format - WHO-WHEN-WHAT-WHERE:

1. WHO: Author's name (surname, followed by initials)

2. WHEN: Year of publication in parentheses

3. WHAT: Title of the article in sentence case

4. WHERE: Journal title (italicized, title case), volume, issue, pages, DOI

Article titles should be in sentence case meaning only the first word of the title and subtitle and any proper nouns are capitalized.

Journal titles should be italicized and in title case meaning all content words are capitalized.

Multiple Authors Rules:

• Two authors: cite both surnames separated by an ampersand (&)

• More than two authors: cite only the surname of the first author followed by "et al."

• Reference list: For 3 to 20 authors, list all names separated by commas with an ampersand before the final author

When you include a direct quote, cite the author, year, and specific page number.

DOI Formatting: If the DOI is not already presented with the "https://doi.org/" prefix in the original source, you should add it to the beginning of the DOI in your reference list entry.

Reference List Formatting:

• Arrange in alphabetical order by first author's surname

• Double space all entries

• Use hanging indent (first line flush left, subsequent lines indented)

• Exclude URL or database information

Common Don'ts:

• Don't include unnecessary elements in in-text citations (article title, publication date, journal name)

• Don't rely on suggested citation formats from databases

• Don't forget to add the "https://doi.org/" prefix for DOIs

Learn the fundamentals of citing journal articles using APA 7th edition format, including in-text citations and reference list entries.

Video Transcript with Key Points Highlighted

This video will cover the proper way to cite secondary sources in APA 7th edition Style. By the end of the video you will understand what secondary sources and indirect citations are, learn how to cite secondary sources in in-text citations and how to cite secondary sources in your reference list.

Primary vs Secondary Sources:

• A primary source reports original content or firsthand information

• A secondary source refers to content that was first reported in another source

Common Examples of Secondary Sources:

• Textbooks summarizing research findings

• Review articles synthesizing multiple primary studies

• Newspaper or magazine articles reporting on a study

Best Practice Recommendation: When conducting academic research you're advised to locate and read the original source material rather than relying solely on the information presented in a secondary source.

The full citation details provided in the reference list of the secondary source can be used to help you find and access the primary source. You can use tools like OneSearch to locate and retrieve the original source document.

In-Text Citation Format for Secondary Sources:

For example, let's say you're reading an article written by Lyon et al. published in 2014, and they cite a study by Rabbit from 1982. If you were unable to access the original Rabbit study yourself, you would cite it in your paper like this:

(Rabbit, 1982, as cited in Lyon et al., 2014)

Since you did not read the Rabbit study directly, you would cite it as an indirect or secondary source using the format described above. This allows you to acknowledge the original work while also being transparent that you accessed it through a secondary source rather than the primary source itself.

Reference List Rules for Secondary Sources:

Unlike the in-text citation, for the reference list entry you will only include the details of the secondary source that you actually accessed.

This means you should not include the primary source (such as the Rabbit 1982 study) in your reference list since you did not directly consult that original work.

Key Rule Summary:

• In-text citation should acknowledge both the original primary source and the secondary source you used

• Reference list entry should only include details for the secondary source you referenced

Essential APA 7th Edition Rules

In-Text Citations

  • • Use ampersand (&) in parenthetical citations for two authors
  • • Use "et al." for three or more authors
  • • Include page numbers for direct quotes

Title Formatting

  • • Article titles: Sentence case (first word + proper nouns)
  • • Journal titles: Title Case (Each Major Word)
  • • Capitalize first word after colon in titles

DOI Formatting

  • • Always use full URL format: https://doi.org/
  • • Add prefix if missing from original source
  • • No "Retrieved from" needed for DOIs

Secondary Citations

  • • Use "as cited in" connector phrase
  • • Only secondary source appears in reference list
  • • In-text: cite both primary and secondary sources

Author Order

  • • Reference list: alphabetize by first author's surname
  • • Never rearrange author order from original
  • • Use last name, first initial format

Reference List Rules

  • • Only include sources you actually accessed
  • • Use hanging indent for each entry
  • • Italicize journal titles, not article titles
