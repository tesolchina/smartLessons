獎項

傑出少年作家獎： 現金獎港幣1500元，獎狀乙張
少 年 作 家 獎：    獎狀乙張

參賽須知

  1. 全港中學生均可參加。
  2. 作品類別：小說、散文或新詩。
  3. 由學校挑選校内 3 名學生之作品共 3 篇參賽（學生年齡不限、文類不拘）。每名學生只可提交一篇作品。
  4. 每校可獲「少年作家獎」獎項最多一個。
  5. 參賽作品字數 (包括標點符號)：
    小說──不超過2000字
    散文──不超過1000字
    新詩──不超過50行

參賽辦法

  1. 參賽學校須提交作品 3 篇（3名參賽學生，每名學生一篇）。
  2. 報名辦法：(a) 網上報名表格投稿；或 (b) 電郵投稿。
    恕不接受郵寄或傳真投稿。
    1. 網上報名表格投稿
      填寫網上報名表格，並上載以下文件：
      1. 每篇參賽作品 WORD 檔 或 PDF 檔；及
      2. 由校方填寫及蓋章的「學校證明書」JPG檔 或 PDF檔。如無學校蓋章，恕不處理。
        網上報名表
        「學校證明書」下載
    2. 電郵投稿
      請將以下文件電郵至lcchi@hkbu.edu.hk，電郵註明「少年作家獎」及學校名稱：
      1. 每篇參賽作品 WORD 檔 或 PDF 檔；及
      2. 由校方填寫及蓋章的「參賽表格」 JPG 檔或 PDF 檔。如無學校蓋章，恕不處理。
        參賽表格下載
  3. 「學校證明書」及「參賽表格」可於比賽網頁下載。
  4.  學生個人資料應填寫在參賽表格上。請勿在作品中寫上個人或學校資料（包括姓名、學校名稱及班別）。
     

來稿須知

  1. 參賽作品中的字號不可以小於12點，雙行行距。
  2. 參賽作品必須寫明題目，每頁清楚標明頁碼，於正文末處列明總字數（包括標點符號）。
  3. 參賽作品必須從未以任何形式公開發表（包括文字傳媒和電子傳媒──例如公開或私人網頁），亦未參加其他比賽。
  4. 參賽作品須為個人原創作品，翻譯作品 / 翻譯改寫或改編作品概不接受。

其他須知

  1. 評審過程中，所有參賽作品均糊名處理。
  2. 評審結果以評審委員之決定為準，參賽者不得異議。
  3. 比賽結果公布前，參賽者不可公開發表參賽作品，亦不能以參賽作品參加其他比賽。
  4. 大會擁有參賽作品之使用權，獲獎作品可能編印成書、刊載於香港文學刊物，及/或於頒獎禮上展示。
  5. 得獎名單將於2025年5月公布。所有得獎學校將另函通知。
  6. 頒獎典禮將於2025年6月在香港浸會大學舉行。
  7. 比賽細則若有未盡善處，大會有權修訂。
